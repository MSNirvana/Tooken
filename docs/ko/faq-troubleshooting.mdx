# 常见问题与排查

## API 返回 401

- 检查 `Authorization: Bearer sk-...` 是否正确。
- 确认 Key 未过期、未被撤销。
- 如使用 Claude 兼容接口，可改用 `x-api-key` 头重试。

## API 返回 402

- 账户 크레딧 可能不足，请先充值。
- 如启用 x402 钱包支付，请确认钱包余额与授权额度充足。

## API 返回 429

- 触发限流，建议使用指数退避重试。
- 检查账户套餐对应的 RPM/TPM/并发上限。

## API 返回 502/503

- 通常为上游节点波动，建议快速重试。
- 若持续出现，可切换到 `auto-fast` 或指定备用模型。

## OpenClaw/Claude Code 接入失败

- 检查 Base URL 是否为 `https://api.tooken.ai`（或 `https://api.tooken.ai/v1`）。
- 检查本地环境变量是否已生效（重新打开终端后再试）。

## 仍无法定位问题

请收集以下信息后提交给支持团队：

- 请求时间（含时区）
- 请求路径与模型名
- 返回状态码与错误信息
- 请求 ID（若有）
